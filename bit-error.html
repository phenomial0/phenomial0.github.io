BIT ERROR

Phenomial Digital Art

I forget exactly how these images were generated, except that they arose as a result of binary rounding errors dominating the behaviour of deterministic motion planning trees using very small values of certain parameters. Somehow, despite not using any pseudorandomness, the degenerate tree grew differently for different runs of the same program, which is something I still cannot explain. At the time I thought it was interesting and collected these images before changing the only copy of the code and moving on. NFTs available on Opensea.
